#

| **屬性定義**                           | **家庭圈 ID (Circle ID)**          | **個人配對碼 (Pairing Code)** |
| -------------------------------------- | ---------------------------------- | ----------------------------- |
| 格式規範                               | 6 位英數混合 (大寫英文字母 + 數字) | 4 位純數字                    |
| 範例                                   | K8J2WS                             | 1234                          |
| 持有與產生者                           | 守門人 (建立者)                    |
| 由第一個建立家庭的人產生。             | 守護者 (長輩)                      |
| 在 App 內點擊「顯示配對碼」時產生。    |
| 存續特性                               | 永久固定                           |
| 除非該家庭圈被手動刪除。               | 臨時有效                           |
| 通常設定為 5-10 分鐘或綁定後即失效。   |
| 主要功能                               | 「群組地址」：                     |
| 讓其他家人透過此 ID 加入同一個家庭圈。 | 「授權金鑰」：                     |
| 證明長輩本人同意被該家庭圈監控。       |
| 輸入對象                               | 想加入該家的其他家人。             | 想綁定長輩的守門人。          |

### **1. 守門人端 (家人)：權限管理者**

- **情境 A：進入現有的家**
  - 輸入 **6 位英數家庭圈 ID**。
  - 目的：與其他兄弟姊妹看同一個長輩的狀態。
- **情境 B：綁定新的長輩**
  - 在家庭管理頁面點擊「新增成員」。
  - **輸入 4 位純數字配對碼**。
  - 輸入完畢後，系統會將該長輩的帳號與目前的家庭圈 ID 綁定。

### **2. 守護者端 (長輩)：受保護者**

- **極簡化操作**：
  - 長輩完全**不需要知道**「家庭圈 ID」是什麼。
  - 畫面只有一個大按鈕：**【產生配對碼】**。
  - 點擊後顯示大大的 **1234**。
  - 長輩只需將這 4 個數字告訴家人，剩下的操作（輸入 ID、綁定程序）全部由家人在另一端完成。
